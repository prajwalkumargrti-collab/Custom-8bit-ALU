# Custom 8-bit ALU

An 8-bit Arithmetic Logic Unit (ALU) designed using Logisim Evolution. The ALU performs arithmetic and logical operations including AND, OR, XOR, and ADD.

## Features
- 8-bit input operands
- AND, OR, XOR, and ADD operations
- Multiplexer-based operation selection
- Carry flag generation
- Hexadecimal display output

## Tools Used
- Logisim Evolution

## Future Enhancements
- Subtraction operation
- Zero flag
- Overflow flag
- Register integration
